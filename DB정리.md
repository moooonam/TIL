> ## SQL
<br>

- "Structured Query Language"
- RDBMS(관계형 데이터베이스 관리 시스템)의 데이터를 관리하기 위해 설계된 특수 목적의 프로그래밍 언어 
- SQL은 데이터베이스와 상호작용하는 방법
- SQL commands 종류
  1. DDL(Data Definition Language)
  2. DML(Data Manipulation Language)
  3. DCL(Data Control Language)
